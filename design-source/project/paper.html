Editor's note · A. Reber

A paper that exists because some inks needed somewhere to go. Reserve for the ones whose second colour — the sheen — is the whole point.

Care notes

  • Store sealed→ humidity
  • Keep flat→ curl
  • Don't fold→ creases set
  • Allow 60 s→ wet inks
